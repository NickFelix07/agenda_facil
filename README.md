# AgendaFácil - Sistema de Gerenciamento de Agendamentos

## Como executar o projeto

1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

2. Execute o projeto:
   ```bash
   python run.py
   ```

3. Acesse: http://127.0.0.1:5000

**Primeiro acesso:**
- Vá para /register e crie um usuário.
- Depois faça login.

## Funcionalidades
- Cadastro de Clientes, Serviços e Profissionais
- Agendamentos com validação básica
- Dashboard intuitivo
- Interface responsiva com Bootstrap
